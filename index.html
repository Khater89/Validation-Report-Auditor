Validation rules: IE & FN Status must be OK, IE Due Date must match FN Scheduled Date, IE Due Time must match FN Scheduled Time, and IE FE Name must match FN FE Name.
Exception rules: date mismatches are accepted only when Ticket Type = ITD. time mismatches are accepted when Ticket Type = ITD or ITR.
Action rule: for any row with an open error, click Correct after fixing the source externally. The button turns to Corrected and the row is tracked as completed for statistics and export. No pop-up is used.
